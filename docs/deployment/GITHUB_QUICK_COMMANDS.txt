### Step 2: Run These Commands

**Open Command Prompt in your project folder:**

```bash
cd C:\Users\amaan\OneDrive\Desktop\PROgame
```

**Initialize Git (if not already done):**
```bash
git init
```

**Configure Git (first time only):**
```bash
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
```

**Add all files:**
```bash
git add .
```

**Check what will be uploaded:**
```bash
git status
```

**Commit:**
```bash
git commit -m "Initial commit: CodeLinkers - Gamified Learning Platform"
```

**Add remote (replace 'yourusername'):**
```bash
git remote add origin https://github.com/yourusername/codelinkers.git
```

**Push to GitHub:**
```bash
git branch -M main
git push -u origin main
```

**Done! ✅**

## 🔍 VERIFY BEFORE PUSHING

**Check what will be uploaded:**
```bash
git status
```

**Check if .env is ignored:**
```bash
git check-ignore .env
```
Should output: `.env` ✅

**List all files that will be uploaded:**
```bash
git ls-files
```

**Make sure you DON'T see:**
- ❌ .env
- ❌ Documentation .md files (except README.md)
- ❌ node_modules/

---

## 🚨 IF YOU UPLOADED .ENV BY MISTAKE

```bash
git rm --cached backend/.env
git commit -m "Remove .env file"
git push origin main
```

**⚠️ Then regenerate your API keys immediately!**
