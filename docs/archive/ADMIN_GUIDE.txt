# Admin Panel Guide - Creating Games & Questions

## ✅ Admin Panel is NOW Connected to Database!

The admin panel was previously **static** (showing fake data). Now it's **fully connected** to your MongoDB database!

## 🎮 How to Create Games/Questions

### Step 1: Login as Admin
1. Go to: `http://localhost:5000`
2. Click "Log In"
3. Use admin credentials:
   - **Email**: `admin@codequest.com`
   - **Password**: `admin123`

### Step 2: Access Admin Panel
1. Click "Admin Panel" link in footer
2. Or go directly to: `http://localhost:5000/admin.html`

### Step 3: Navigate to Games Section
1. Click "Games & Questions" in the sidebar
2. You'll see all existing questions from the database

### Step 4: Create New Question
1. Click **"Add New Question"** button (top right)
2. Fill in the form:

#### Required Fields:
- **Question Title**: e.g., "Center a Div with Flexbox"
- **Description**: Explain what the user needs to do
- **Topic**: Choose HTML or CSS
- **Difficulty**: 
  - Beginner (5 points)
  - Pro/Medium (10 points)
  - Ultra Pro (20 points)
- **Expected CSS Solution**: The correct CSS code

#### Optional Fields:
- **Hints**: One hint per line (helps users)
- **Initial HTML Code**: Starting HTML structure
- **Target Output Image**: Upload screenshot of expected result
- **Status**: Active (visible) / Draft (hidden) / Disabled

### Step 5: Submit
1. Click **"Create Question"**
2. Question is saved to MongoDB database
3. Appears immediately in the games list
4. Users can play it right away!

## 🎯 Example: Creating a CSS Game

### Example Question Data:
```
Title: Center a Box with Flexbox
Description: Use CSS Flexbox to center the .box element both horizontally and vertically inside the .container.

Topic: CSS
Difficulty: Beginner
Points: 5 (auto-set based on difficulty)

Hints:
Use display: flex on the container
justify-content: center handles horizontal centering
align-items: center handles vertical centering

Initial HTML:
<div class="container">
    <div class="box"></div>
</div>

Expected CSS Solution:
.container {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 200px;
}

.box {
    width: 80px;
    height: 80px;
    background: linear-gradient(135deg, #6366f1, #ec4899);
    border-radius: 12px;
}

Status: Active
```

## 🔍 How It Works

### When Admin Creates Question:
1. Admin fills form in admin panel
2. Clicks "Create Question"
3. **admin.js** calls `API.Questions.create(questionData)`
4. **api.js** sends POST request to backend
5. **Backend** saves to MongoDB
6. **Backend** returns success response
7. Admin panel shows success notification
8. Question list refreshes automatically

### When User Plays Game:
1. User clicks "Start Playing" on games.html
2. Redirects to `play.html?topic=css&level=beginner`
3. **game.js** calls `API.Questions.getAll()`
4. **Backend** fetches questions from MongoDB
5. Filters by topic & difficulty
6. Loads first question
7. User writes code and submits
8. **Backend** validates answer
9. Awards points if correct

- Test the expected solution yourself
- Use realistic examples

### For Target Images:
- Take screenshot of expected output

### For Initial HTML:
- Keep it simple

- Add classes for styling

## 🐛 Troubleshooting

### "Error creating question"
- Check if you're logged in as admin
- Verify all required fields are filled
- Check browser console for errors
- Ensure backend server is running

### "No questions available"
- Check if questions exist in database
- Verify topic and difficulty match
- Check question status is "active"
- Run seed script to add sample questions

### Questions not appearing in game
- Refresh the page
- Check question status (must be "active")
- Verify topic and difficulty are correct
- Check browser console for errors

## 📞 Need Help?

1. Check browser console (F12)
2. Check backend server logs
3. Verify MongoDB is running
4. Check if admin user has correct role
5. Try running seed script again
