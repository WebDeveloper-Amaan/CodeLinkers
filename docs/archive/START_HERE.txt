## 🚀 START IN 3 STEPS

### Step 1: Install Prerequisites (10 minutes)

**A. Install MongoDB**
- Download: https://www.mongodb.com/try/download/community
- Install with default settings
- It will run automatically

**B. Install Node.js**
- Download: https://nodejs.org/ (LTS version)
- Install with default settings

### Step 2: Install Dependencies (2 minutes)

Open Command Prompt and run:
```bash
cd C:\Users\amaan\OneDrive\Desktop\PROgame\backend
npm install
```

Wait for installation to complete.

### Step 3: Start the Server (1 minute)

**Option A: Quick Start (Easiest)**
- Double-click `START_SERVER.bat` in the PROgame folder

**Option B: Manual Start**
```bash
cd C:\Users\amaan\OneDrive\Desktop\PROgame\backend
npm run seed    # Load sample data (first time only)
npm start       # Start server
```

## 🐛 TROUBLESHOOTING

### Server won't start?
- Make sure MongoDB is running
- Check if port 5000 is free
- Run `npm install` in backend folder

### Can't connect to database?
- Verify MongoDB is installed
- Check if MongoDB service is running
- Look for errors in terminal

### Page not loading?
- Check if server is running
- Try: http://localhost:5000/index.html
- Clear browser cache
