# 🔗 Google Drive Link - Quick Reference

## Step 1: Upload to Google Drive
1. Go to https://drive.google.com
2. Click "New" → "File upload"
3. Select your PDF file
4. Wait for upload to complete

## Step 2: Get Shareable Link
1. Right-click on uploaded file
2. Click "Share"
3. Click "Change to anyone with the link"
4. Click "Copy link"

## Step 3: Convert Link Format

### Original Link (from Google Drive):
```
https://drive.google.com/file/d/1ABC123XYZ456/view?usp=sharing
```

### Extract the FILE_ID:
```
1ABC123XYZ456
```
(The part between `/d/` and `/view`)

### Convert to Direct Download Link:
```
https://drive.google.com/uc?export=download&id=1ABC123XYZ456
```

## 📋 Quick Formula

```
Original:  https://drive.google.com/file/d/FILE_ID/view?usp=sharing
Converted: https://drive.google.com/uc?export=download&id=FILE_ID
```

## ✅ Example

**Original:**
```
https://drive.google.com/file/d/1a2b3c4d5e6f7g8h9i0j/view?usp=sharing
```

**Converted:**
```
https://drive.google.com/uc?export=download&id=1a2b3c4d5e6f7g8h9i0j
```

## 🎯 Use This Link in Admin Panel

Paste the **converted** link in the "Google Drive Link" field when adding notes.

## 💡 Pro Tips

1. **Keep files organized** - Create folders in Google Drive for different categories
2. **Use descriptive names** - Name files clearly (e.g., "BCA-Sem3-DataStructures.pdf")
3. **Test the link** - Open converted link in browser to verify it downloads
4. **Batch upload** - Upload multiple files at once, then get links for each

## 🔒 Privacy Note

- Links set to "anyone with the link" are publicly accessible
- Anyone with the link can download the file
- Perfect for educational content sharing
- No Google account required for users to download
